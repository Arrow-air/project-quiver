# 3D Printing

Slicer profiles and print settings for 3D-printed Quiver parts.

## 3D-printed parts

The following BOM subcategories contain 3D-printed components (PETG-CF
unless noted otherwise):

| BOM | Subcategory | Parts |
|---|---|---|
| 1300 | Landing Gear | Tube joints |
| 2100 | Attachment Interface | Spacers |
| 2200 | Battery Slider | Battery sliders |
| 2300 | Equipment Mount | PCB mounts, sensor mounts, camera mount |
| 2400 | Cockpit Enclosure | Main enclosure, top cap, anchors, clips |

## Organization

Store slicer profiles and settings grouped by printer or material. For
example:

```
printing/
    bambu_x1c_petg_cf.json
    prusa_mk4_petg_cf.3mf
```
